# Product Sales Analysis using MS Excel

## Overview
Analyzed retail sales data to extract business insights using Excel. The project covers data cleaning, pivot tables, charts, and dashboards.

## Dataset
Contains sales, discounts, delivery, customer type, and regional information.

## Key Analysis
- Sales by Region and Product
- Monthly Sales Trends
- Salesperson Performance
- Customer Type and Store Location Analysis
- Returned Items Analysis
- Delivery Time Analysis

## Tools
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard
